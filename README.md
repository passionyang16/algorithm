# 알고리즘 문제 풀이

## 문제 유형
- 해시 (Hash)
- 스택 (Stack) / 큐 (Queue)
- 힙 (Heap)
- 정렬 (Bubble Sort, Insertion Sort, Merge Sort, Quick Sort, Selection Sort)
- 완전탐색 (Search) / 이분탐색 (Binary Search)
- 너비우선탐색 (BFS) / 깊이우선탐색 (DFS)
- 다익스트라 (Dijkstra) / 크루스칼 (Kruskal) / 프림 (Prim)
- 동적계획법 (Dynamic Programming)
- 탐욕법 (Greedy)
- 재귀 (Recursive)
- 백트레킹 (BackTracking)
